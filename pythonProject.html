Python Project

Write-Up

I experienced difficulties while developing my Python Project, though I resolved them and incorporated them as part of an incremental and iterative development. My first difficulty was when I tried to call a function inside of another function. I had personally written both functions but I had forgotten to pass in the variables inside of the parentheses. My groupmates reminded me and this problem was resolved. Another problem I encountered while coding my Python Project was telling the user that they had won the game and breaking the while loop after the user guessed all of the correct letters. Our code could tell when the user had lost but not when they had won. We eventually solved this issue by assigning the guesses their own variable and comparing it to the answer. I realized that we were unintentionally making the code unnecessarily complicated and once we simplified it, it worked much better. As part of an incremental and iterative development, I experienced difficulties while developing my Python Project, though as a group we worked through and resolved them.
